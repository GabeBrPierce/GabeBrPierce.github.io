Single Incursive Functions

Intro

Google Sheets is a Robust and complex software. Ranging from simple and intuitive formatting to advanced and complex function nesting. I have grown to love the software, and I love it even more since I discovered this neat trick.

Getting Started

First, you need to enable incusive calculation, then you need to set the number of calculations. The default as of 9/10/2021 is 50. Change this value to 1. Then click Save Settings.

Okay now what?

Now here is where things get interesting, now you can reference a cell in it's own formula! A easy way to show you its power is to develop a counter.

First I am going to format that speadsheet we set up eariler. Here is how it is going to look.

Now lets add a function that references itself! Here is what I came up with.

=IF(RESET, 0, IF(UP_CHECKBOX,COUNTER+1,IF(DOWN_CHECKBOX,COUNTER-1,COUNTER)))

A2-4 is the counter, and cells B3 and B5 are the up and down counters, respectifully. Lets see how it works!
